Imagens no HTML

Para inserir uma imagem no HTML, utilizamos a tag img. A tag img é uma tag vazia, ou seja, não possui tag de fechamento. A tag img possui dois atributos obrigatórios: src e alt. O atributo src é o caminho da imagem que será exibida na página. O atributo alt é o texto alternativo que será exibido caso a imagem não possa ser carregada.
